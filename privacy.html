Privacy Policy

Last updated: 8 June 2026

This Privacy Policy explains how the Discord bot collects and uses data.

1. Data Collected

The Bot may collect only the minimum data required for functionality:

  • Discord User IDs (for ticket systems and permissions)
  • Discord Server (Guild) IDs
  • Channel IDs (for ticket creation and setup)
  • Configuration data (such as welcome message settings)

2. Messages

The Bot does not read or store normal user messages or private messages. It only processes messages related to commands or ticket interactions.

3. How Data is Used

  • To provide ticket and support systems
  • To manage welcome messages
  • To store server settings
  • To prevent abuse

4. Data Sharing

No data is sold or shared with third parties. Data is only accessible by the bot owner.

5. Data Storage

Data is stored only as long as necessary for functionality. Server data can be removed upon request.

6. Security

We take reasonable steps to protect data, but no system is 100% secure.
