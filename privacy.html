Privacy Policy

Last updated: 8 June 2026

This Privacy Policy explains how the Discord bot handles data.

1. Data Collected

The Bot may collect only the data required for functionality:

  • Discord User IDs (for ticket systems and permissions)
  • Discord Server IDs
  • Channel IDs (for ticket channels and setup)
  • Basic configuration data (such as welcome settings)

2. Messages

The Bot does NOT read or store private messages or general user messages. It only interacts with messages required for ticket creation or bot commands.

3. How Data is Used

  • To provide ticket and support system functionality
  • To manage welcome messages and server setup features
  • To prevent abuse and ensure proper bot operation

4. Data Sharing

No data is sold or shared with third parties. Data is only accessible by the bot owner.

5. Data Storage

Data is stored only as long as needed for bot functionality. Server owners may request data removal.

6. Security

Reasonable steps are taken to protect stored data, but no system is 100% secure.
